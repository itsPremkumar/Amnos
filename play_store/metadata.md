# Play Store Metadata for Amnos

## 📝 App Metadata
**Title**: Amnos
**Short Description**: Privacy-focused, hardened Android browser.
**Full Description**: 
Amnos is a high-security, privacy-hardened Android browser built on WebView. It features aggressive tracker blocking, WebRTC shutdown, finger-print obfuscation, and a zero-persistent-footprint model. Designed for local-only browsing sessions with maximum privacy.
